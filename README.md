## ğŸ§ª How to Run Locally

1. **Clone the Repository**

```bash
git clone https://github.com/iaadillatif/techxninjas.git
cd techxninjas
```

2. **Install Dependencies**

```bash
npm install
```

Create a `.env.local` file and paste your Supabase/EmailJS keys as described

```bash

```
